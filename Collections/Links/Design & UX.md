## Assets

- [Animated Icons](https://github.com/pqoqubbw/icons) - A collection of smooth animated icons for React & TypeScript projects built with Framer Motion and Lucide.
- [Fontshare](https://www.fontshare.com/) - Quality fonts. Free. No gimmicks. Curated by Indian Type Foundry.
- [Geist](https://vercel.com/font) - Geist Sans & Mono. The Vercel font.
- [Heroicons](https://heroicons.com/) - Beautiful hand-crafted SVG icons, by the makers of Tailwind CSS.
- [Inter](https://rsms.me/inter/) - The standard for UI fonts. Highly legible text font for computer screens.
- [Lucide Icons](https://lucide.dev/) - Beautiful & consistent icons.
- [Phosphor Icons](https://phosphoricons.com/) - A flexible icon family for interfaces, diagrams, presentations — whatever, really.
- [Remix Icon](https://remixicon.com/) - Open-source neutral-style system symbols. Elaborately crafted for designers and developers.
- [Simple Icons](https://simpleicons.org/) - Over 3000 Free SVG icons for popular brands.
- [Tabler Icons](https://tabler.io/icons) - Over 5200 pixel-perfect icons for web design.
- [Uncut.wtf](https://uncut.wtf/) - A curated catalog of beautiful typefaces, free for commercial use.
- [Unsplash](https://unsplash.com/) - The internet’s source of freely-usable images.
- [Velvetyne](https://velvetyne.fr/) - Designing and distributing free and open source typefaces since 2010. Experimental & Wild.

## Tools

- [Coolors](https://coolors.co/) - The super fast color palettes generator.
- [Excalidraw](https://excalidraw.com/) - Virtual whiteboard for sketching hand-drawn like diagrams.
- [Khroma](https://www.khroma.co/) - The AI color tool for designers. Learn from your favorites.
- [Open Color](https://yeun.github.io/open-color/) - Open source color scheme for UI design.
- [Penpot](https://penpot.app/) - Open-source design and prototyping platform meant for cross-domain teams. alternative to Figma.
- [Radix Colors](https://www.radix-ui.com/colors) - An open-source color system for designing beautiful, accessible websites.
- [tldraw](https://tldraw.com/) - A tiny little drawing app.
