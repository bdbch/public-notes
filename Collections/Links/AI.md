## Applications

- [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) - A comprehensive collection of battle-tested configuration files, agents, and skills for Claude Code.
- [KittenTTS](https://github.com/KittenML/KittenTTS) - Ultra-compact, CPU-optimized text-to-speech model under 25MB for lightweight deployment.
- [Open WebUI](https://github.com/open-webui/open-webui) - User-friendly WebUI for LLMs (Formerly Ollama WebUI).

## Assistants

- [Cline](https://github.com/cline/cline) - Autonomous coding assistant for VS Code that can edit files, run commands, and use the browser.
- [Continue](https://github.com/continuedev/continue) - Open-source AI code assistant. Connect any model (local or remote) and build your own custom autocomplete and chat experiences.

## Inference

- [LocalAI](https://github.com/mudler/LocalAI) - The free, Open Source OpenAI alternative. Self-hosted, community-driven and local-first.
- [Ollama](https://ollama.com/) - Get up and running with large language models (like Llama 3, Mistral, Gemma) locally.
