## Assets

- [Itch.io](https://itch.io/game-assets) - Indie game assets marketplace. Home to thousands of free and paid assets.
- [Kenney](https://kenney.nl/assets) - Thousands of completely free game assets (CC0). The 'Asset Jesus'.
- [OpenGameArt](https://opengameart.org/) - The definitive repository for open source game art.

## Engine

- [Bevy](https://bevyengine.org/) - A data-driven game engine built in Rust. Simple, Modular, Fast.
- [Defold](https://defold.com/) - The ultimate engine for 2D games. Performance and stability out of the box.
- [Godot](https://godotengine.org/) - The feature-packed, cross-platform game engine to create 2D and 3D games from a unified interface.
- [Raylib](https://www.raylib.com/) - A simple and easy-to-use library to enjoy videogames programming.

## Tools

- [LDtk](https://ldtk.io/) - Modern 2D level editor from the creator of Dead Cells.
- [Material Maker](https://www.materialmaker.org/) - A procedural texture generation tool based on the Godot Engine.
- [Ogmo Editor](https://ogmo-editor-3.github.io/) - A generic visual level editor for any game engine.
- [Pixelorama](https://github.com/Orama-Interactive/Pixelorama) - A free & open-source 2D sprite editor, made with the Godot Engine.
- [Tiled](https://www.mapeditor.org/) - Your free, easy to use and flexible level editor.
