## Core Utils

- [bat](https://github.com/sharkdp/bat) - A `cat` clone with syntax highlighting and Git integration.
- [dust](https://github.com/bootandy/dust) - A more intuitive version of du in rust.
- [eza](https://github.com/eza-community/eza) - A modern, maintained replacement for `ls`.
- [fd](https://github.com/sharkdp/fd) - A simple, fast and user-friendly alternative to 'find'.
- [ripgrep](https://github.com/BurntSushi/ripgrep) - A line-oriented search tool that recursively searches the current directory for a regex pattern. Faster than grep.
- [zoxide](https://github.com/ajeetdsouza/zoxide) - A smarter cd command. It remembers which directories you use most frequently, so you can \"jump\" to them in just a few keystrokes.

## Dev Utilities

- [K9s](https://k9scli.io/) - Kubernetes CLI To Manage Your Clusters In Style.
- [Pandoc](https://pandoc.org/) - A universal document converter.
- [tokei](https://github.com/XAMPPRocky/tokei) - Count your code, quickly.

## File Management

- [Broot](https://dystroy.org/broot/) - A better way to navigate directory trees.
- [Yazi](https://yazi-rs.github.io/) - Blazing fast terminal file manager written in Rust, based on async I/O.

## Git

- [Delta](https://github.com/dandavison/delta) - A viewer for git and diff output.
- [Lazygit](https://github.com/jesseduffield/lazygit) - Simple terminal UI for git commands.

## Media

- [FFmpeg](https://ffmpeg.org/) - A complete, cross-platform solution to record, convert and stream audio and video.

## Shell

- [Atuin](https://atuin.sh/) - Magical shell history. Sync, search, and backup shell history.
- [Starship](https://starship.rs/) - The minimal, blazing-fast, and infinitely customizable prompt for any shell.
- [Zellij](https://zellij.dev/) - A terminal workspace with batteries included. easier than tmux.

## Terminal

- [Alacritty](https://alacritty.org/) - A cross-platform, OpenGL terminal emulator.
- [Ghostty](https://ghostty.org/) - A fast, native, feature-rich terminal emulator optimizing for performance and platform-native feel.
- [WezTerm](https://wezfurlong.org/wezterm/) - A GPU-accelerated cross-platform terminal emulator and multiplexer.
